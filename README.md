# Framelet

## Install

Download the latest Apple Silicon build from [GitHub Releases](https://github.com/mcxen/Framelet/releases/latest), or install it with Homebrew:

```bash
brew tap mcxen/framelet https://github.com/mcxen/Framelet
brew install --cask framelet
```

Upgrade a Homebrew installation with `brew upgrade --cask framelet`. Framelet can also check GitHub Releases from **Settings → About**; installations in a non-writable location should be upgraded through Homebrew instead.


Framelet is a native macOS lossless trimming tool focused on opening media,
marking precise ranges, organizing segments, saving projects, and exporting
segments through FFmpeg stream copy.

`.codex/environments/environment.toml`.

## Current Build

- SwiftPM macOS 14+ SwiftUI app.
- Native `AVPlayerView` playback with custom transport controls.
- FFprobe JSON media analysis.
- Project save/load as `.frameletproject` JSON.
- In/out point marking and multiple segment creation.
- Segment selection, editing, deletion, enable toggles, and stream selection.
- Segment CSV import/export compatible with the common `Start,End,Name` layout,
  including marker-only CSV rows that are converted to continuous ranges.
- Timeline drawing with playhead, in/out markers, segment hit testing, edge
  dragging for trim adjustments, and drag-to-reorder.
- FFprobe keyframe scanning with timeline markers and per-segment keyframe
  offset diagnostics.
- AVFoundation thumbnail generation with a visual thumbnail strip in the
  timeline.
- FFmpeg audio waveform extraction with a waveform strip in the timeline.
- FFmpeg-assisted proxy preview inspired by LosslessCut: create a compatible
  low-resolution MP4 preview while keeping exports based on the original media.
- LosslessCut-style last FFmpeg/FFprobe command log with copy and clear actions.
- LosslessCut-inspired timeline zoom window with zoom in/out, fit, and horizontal
  panning.
- Separate lossless segment export using `ffmpeg -c copy`.
- Merged export through temporary segment files and FFmpeg concat demuxer.
- Optional video picture cropping with a player overlay preview. Crop export uses
  FFmpeg's `crop` filter and re-encodes video while copying other selected
  streams where possible.

## Requirements

Install FFmpeg so `ffmpeg` and `ffprobe` are available at one of:

- `/opt/homebrew/bin`
- `/usr/local/bin`
- `/usr/bin`

Later app packaging can place the executables in `Resources/Tools`.

## Releases

Push a semantic version tag to trigger the GitHub Actions release workflow:

```bash
git tag v0.1.0
git push origin v0.1.0
```

The workflow builds an Apple Silicon macOS application, packages it as a ZIP,
and publishes it on the corresponding GitHub Release. The current artifact is
ad-hoc signed and is not notarized.

## Known Follow-ups

- Merged export now works for compatible stream layouts generated by Framelet's
  segment step. More detailed preflight diagnostics are still needed for unusual
  stream selections or container-specific subtitle/data streams.
- Automatic playback fallback and richer export diagnostics are scaffolded for
  future work but not complete in this MVP.
- Security-scoped bookmark persistence is stored, but full sandbox entitlement
  packaging is not configured yet.
